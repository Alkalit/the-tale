Механизм политики позволяет игрокам оказывать прямое влияние на мир игры путём прямого демократического голосования — всё как в лучших сказочных странах.

Каждый игрок может создать законопроект, который вступит в силу, если по истечении заданного промежутка времени наберёт поддержку других игроков. Законопроекты позволяют менять ту часть мира, на которую не имеют прямого влияния герои: переименовывать города, изменять их население. Для закона создаётся отдельная тема на форуме, в которой можно вести обсуждения. Во время голосования автор закона может его отредактировать (например, по итогам обсуждения), в этом случае будут сброшены все голоса.

Игрок может голосовать «за», «против»

Правила

  • каждый игрок может выдвинуть законопроект, для обсуждения законопроекта создаётся отдельная тема на форуме;
  • игрок может иметь только один активный законопроект, пока по его предложению идёт голосование игрок не может предложить ещё один закон;
  • агитационные компании игроки, по желанию, могут проводить самостоятельно, пользуясь доступными средствами;
  • игроки могут голосовать «за», «против» либо «воздержаться». Изменить своё решение нельзя — делайте выбор аккуратно;
  • в подведении итогов учитываются только голоса отданые «за» или «против»;
  • до окончания голосования точное количество голосов отданых «за» или «против» скрыто;
  • на голосование отводится строго определённое количество дней: {{(bills_settings.BILL_LIVE_TIME/24/60/60)|int}};
  • если во время голосования автор закона его отредактирует, будут сброшены все результаты голосования и сроки;
  • для принятия законопроекта, за время голосования надо преодолеть два барьера: количественный и процентный;
  • количественный барьер — законопроект должен набрать минимально допустимое количество голосов «за»;
  • процентный барьер — количество голосов «за» должно быть больше минимально допустимого процента (от всех голосов);
  • каждый законопроект должен пройти проверку модератора, главное требование — соблюдение общепринятых культурных норм — запрещаются любого рода оскорбления, нарушения законодательства и аналогичные нарушения;
  • по истечении времени голосования закон будет либо отклонён, либо применён (сразу или в ближайшее возможное время).

Ограничение голосования

По некоторым типам законов могут голосовать не все игроки.

Если закон как-либо связан с городами, то голосовать по нему могут только игроки, активно помогающие хотя бы одному городу, связанному с этим законом. Узнать, каким городам помогает игрок, можно на его странице. Помощью считается выполнение задания в пользу персонажа, проживающего в городе. За помощь каждому персонажу начисляется 1 очко помощи. Игрок может голосовать за первые {{bills_settings.PLACES__TO_ACCESS_VOTING}} городов из списка, отсортированного по величине помощи, и за города, имеющие такое же число помощи, как у {{bills_settings.PLACES__TO_ACCESS_VOTING}}-ого города.

Информация о помощи не хранится вечно, учитываются только последние {{heroes_settings.PLACE_HELP_HISTORY_SIZE}} случаев оказания помощи.
